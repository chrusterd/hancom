# MORNING SLOT 웹사이트 기획서

## 1. 프로젝트 개요

MORNING SLOT은 사용자가 하루를 시작하기 전, 슬롯머신을 돌려 오늘의 컬러, 아침 메뉴, 음악 장르를 무작위로 추천받는 인터랙티브 웹사이트다. 단순 추천 도구가 아니라 카지노 슬롯머신의 재미와 아침 하늘의 산뜻한 분위기를 결합해, 가볍고 즐거운 아침 루틴 경험을 제공한다.

## 2. 기획 의도

아침마다 무엇을 입을지, 무엇을 먹을지, 어떤 음악을 들을지 고민하는 사용자를 위해 선택의 부담을 줄이고 작은 재미를 제공한다. 슬롯머신이라는 직관적인 장치를 활용해 사용자가 버튼을 누르는 순간 기대감, 결과 확인, 기록 저장까지 이어지는 짧은 경험 흐름을 만든다.

## 3. 주요 사용자

- 매일 아침 가볍게 기분 전환이 필요한 사용자
- 의상, 식사, 음악을 랜덤 추천받고 싶은 사용자
- 짧은 인터랙션과 시각적 재미를 선호하는 사용자
- 모바일에서 간단히 사용할 수 있는 미니 웹 서비스를 원하는 사용자

## 4. 핵심 콘셉트

### 키워드

- 아침 루틴
- 슬롯머신
- 랜덤 추천
- 잭팟 이벤트
- 기록 아카이브
- 산뜻한 하늘 그래픽

### 디자인 방향

- 파우더 블루, 스틸 블루, 샴페인 골드를 중심으로 한 세련된 아침 카지노 톤
- 2D 감성의 아침 하늘 배경 그래픽
- 부드러운 구름, 햇살, 새 실루엣 애니메이션
- 과도한 게임 UI보다 아침에 어울리는 차분하고 밝은 분위기 유지

## 5. 화면 구성

## 5.1 홈 화면

홈 화면은 서비스의 핵심 경험이 바로 시작되도록 구성한다.

### 구성 요소

- 아침 하늘 애니메이션 배경
- 서비스 타이틀 `MORNING SLOT`
- 오늘의 잭팟 확률 정보
- 3개의 슬롯 릴
  - 컬러
  - 아침 메뉴
  - 음악 장르
- 레버형 실행 버튼
- 결과 요약 카드
- 지금까지 나온 조합 기록
- footer 영역

## 5.2 슬롯머신 영역

슬롯머신은 세 개의 릴로 구성된다. 각 릴은 3D 회전처럼 보이는 애니메이션으로 움직이며, 멈춘 뒤 당첨된 항목이 중앙에 표시된다.

### 릴 항목

컬러:

- 빨강
- 파랑
- 초록
- 베이지
- 검정
- 흰색
- 노랑
- 보라

아침 메뉴:

- 토스트
- 시리얼
- 김밥
- 과일
- 요거트
- 죽
- 샌드위치
- 블랙커피

음악 장르:

- 재즈
- 시티팝
- 클래식
- 로파이
- 케이팝
- 록
- 발라드
- 무음

## 5.3 결과 요약 영역

슬롯머신이 멈춘 뒤 홈 화면 안에서 결과를 바로 확인할 수 있다.

### 표시 정보

- 오늘의 조합
- 현재 위치 기반 장소 정보
- 아침, 점심, 저녁 날씨 정보
- 추천 조합 카드
- 추천 코멘트

## 5.4 뽑은 내역

사용자가 뽑은 조합은 브라우저 localStorage에 저장된다. 홈 화면에서는 최근 10개의 조합을 보여준다.

### 표시 정보

- 조합 제목
- 컬러, 메뉴, 음악 조합
- 날짜와 시간
- 잭팟 여부

잭팟 항목은 일반 항목보다 눈에 띄도록 골드 톤 배경, 강조 테두리, `JACKPOT` 배지를 적용한다.

## 5.5 footer

사이트 하단에는 저작권과 외부 링크를 제공한다.

### 구성

- Copyright
- Blog 링크
- GitHub 링크

## 6. 주요 기능

## 6.1 슬롯머신 실행

사용자가 레버 버튼을 누르면 세 개의 릴이 순차적으로 회전한다. 각 릴은 다른 종료 타이밍을 가져 실제 슬롯머신처럼 기대감을 만든다.

### 동작 흐름

1. 사용자가 레버 버튼 클릭
2. 버튼 비활성화
3. 슬롯 릴 회전 시작
4. 결과 조합 결정
5. 릴이 순차적으로 멈춤
6. 결과 카드 표시
7. 기록 저장
8. 버튼 재활성화

## 6.2 잭팟 시스템

일정 확률로 사전에 정의된 특별 조합이 등장한다. 현재 잭팟 확률은 50%로 설정되어 있다.

### 잭팟 연출

- 잭팟 전용 결과 문구
- 잭팟 팝업 텍스트
- 색종이 confetti 효과
- 잭팟 사운드
- 슬롯머신 flash 효과
- 당첨 항목 이모지 확대
- 뽑은 내역 강조 표시

## 6.3 날씨 및 위치 정보

사용자의 브라우저 위치 권한을 활용해 현재 지역명을 가져오고, 날짜 기반으로 아침, 점심, 저녁 날씨를 표시한다.

### 모바일 대응

- 장소는 상단 전체 폭으로 표시
- 아침, 점심, 저녁 날씨는 세로 카드로 정렬
- 시간대 라벨은 작은 배지 형태로 표시
- 긴 장소명은 줄바꿈 가능

## 6.4 기록 저장

결과는 localStorage에 저장된다.

### 저장 데이터

- 생성 일시
- 컬러
- 메뉴
- 음악
- 제목
- 코멘트
- 잭팟 여부
- 잭팟 이름

저장된 내역은 최신순으로 표시되며, 화면에는 최근 10개가 노출된다.

## 8. 반응형 전략

모바일에서도 핵심 경험이 유지되도록 다음 전략을 적용한다.

- 슬롯머신 세 릴은 모바일에서도 가로 3열 유지
- 릴, 패딩, gap, 이모지 크기를 모바일에 맞게 축소
- 날씨 정보는 세로 카드 구조로 전환
- 기록 항목은 모바일에서 세로 정렬
- footer 링크는 줄바꿈 가능한 flex 구조 사용

## 11. 기대 효과

MORNING SLOT은 사용자가 아침을 시작하기 전 짧은 시간 안에 재미와 선택지를 동시에 얻을 수 있는 웹 경험을 제공한다. 슬롯머신의 기대감, 잭팟의 보상감, 아침 하늘 그래픽의 산뜻함이 결합되어 반복 방문을 유도하는 미니 인터랙티브 서비스로 확장할 수 있다.
